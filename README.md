## 🚀 How to Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/ixdbl79/shadoshka-docs.git
   cd shadoshka-docs
   ```

2. **Open with Live Server** (recommended)
   - In VSCodium/VS Code, right-click `index.html`
   - Select **"Open with Live Server"**

3. **Or use Python HTTP Server**
   ```bash
   python3 -m http.server 8000
   ```
   Then open `http://localhost:8000`

---

## 📝 How to Add a New Page

1. Copy `blank.html` to a new file (e.g., `new-page.html`)
2. Add your content in the `<main>` section
3. The header and footer will load automatically via `js/main.js`
4. Link to it from `index.html` or other pages
